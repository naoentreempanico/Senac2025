Trabalhando com ID e Classes

Seletores ID

É usado um sletor de id em um elemnto HTML para deixar ele unico e especifico

Deve ser usado uma única vez na página.

Exemplo

USANDO UM ID

Seletores de classe

O seletor de classe seleciona elemnto com um atributo de classe especifico.

Pode ser usando mais de uma vez na página.

Exemplo

USANDO CLASSE

USANDO CLASSE

USANDO CLASSE

USANDO TEXTO
